# Getting Started

Sign up for Kropply using your email at **<a href="https://www.kropply.com" target="_blank">Kropply.com</a>**.
You will receive a token through the your email.

> [!IMPORTANT]
> Make sure that noreply@tm.kropply.com is whitelisted.

### Installing

<Tabs>
  <TabItem value="marketplace" label="Visual Studio Marketplace" default>
    Install Kropply using the VS Code Marketplace <a href="https://marketplace.visualstudio.com/items?itemName=kropply.kropply" target="_blank">here!</a>.
  </TabItem>
  <TabItem value="extensions" label="Extensions">
    1. Open the Extensions Tab in Visual Studio Code
    2. Search for "Kropply"
    3. Click Install

    <!-- TODO change the downloand picture here -->
    <img src="/img/download-kropply.png" style={{maxWidth: '60%'}}/>

  </TabItem>
</Tabs>

### Logging In

> Once Kropply is downloaded, open the extension by clicking on the Kropply logo `>>>`.

<img src="/img/left-panel.png" style={{ maxWidth: "60%" }} />

> Click the **Set access token** button shown below to get logged in.

<img src="/img/main-panel-ready.png" style={{ maxWidth: "60%" }} />

> A popup will appear where you can paste in your token.

<img src="/img/token-box.png" style={{ maxWidth: "60%" }} />

> You should now see that you are connected at the bottom right of VS Code.

<img src="/img/connected-to-kropply.png" style={{ maxWidth: "60%" }} />
