# Scenarios

Written in TypeScript, but compatible with **your language**:
- Python
- SQL
- C#
- C++
- R
- TypeScript
- JavaScript
- Java

### When to use it?

Anytime you're writing code!

When using our terminal, Kropply will automatically run in the background, letting you know where you can fix your code.
